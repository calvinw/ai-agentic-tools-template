# GitHub Pages Setup

This skill helps with setting up and managing GitHub Pages for your projects.

## Usage

Use this skill to:
- Initialize a GitHub Pages repository
- Configure GitHub Pages settings
- Deploy static sites to GitHub Pages
- Manage GitHub Pages branches and configurations

## Features

- Guides through GitHub Pages setup process
- Provides configuration templates
- Helps with custom domain setup
- Assists with theme selection and customization
- Troubleshoots common GitHub Pages issues

## Parameters

- **repository-name** (optional): Name of the repository
- **template-type** (optional): Type of GitHub Pages template to use (jekyll, static, etc.)
- **custom-domain** (optional): Custom domain for the site

## Output

Setup instructions and configuration guidance for GitHub Pages deployment.
